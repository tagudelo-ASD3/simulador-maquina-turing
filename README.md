# 🧠💻 Simulador de Máquina de Turing

¡Bienvenido al **Simulador de Máquina de Turing**!
Este proyecto es una aplicación web interactiva que te permite experimentar, visualizar y aprender cómo funciona una máquina de Turing paso a paso.

---

## 🚀 ¿Qué es esto?

Una **máquina de Turing** es un modelo matemático fundamental en la computación teórica. Este simulador te permite:

- 🟢 Ejecutar algoritmos clásicos como suma binaria y escritura de cadenas.
- 📝 Visualizar la cinta, el cabezal y los estados en tiempo real.
- 🦾 Entender el funcionamiento interno de los autómatas y la lógica de transición de estados.

---

## 🛠️ Tecnologías utilizadas

- **Frontend:** Angular + TypeScript
- **Backend:** FastAPI (Python)
- Comunicación vía API REST

---

## ✨ Características principales

- 🎬 Animación paso a paso de la ejecución de la máquina.
- 🧩 Selección de algoritmos (suma binaria, escribir nombre, etc).
- 🖱️ Interfaz intuitiva y moderna.
- 📊 Visualización de la cinta, el cabezal y la tabla de transición.
- ⚡ Velocidad ajustable y control manual de la simulación.

---

## 📦 Instalación y uso rápido

1. **Clona el repositorio:**

   ```bash
   git clone https://github.com/tu-usuario/simulador-maquina-turing.git
   cd simulador-maquina-turing
   ```

2. **Backend (FastAPI):**

   ```bash
   cd turing-simulator/backend
   pip install -r requirements.txt
   uvicorn main:app --reload
   ```

3. **Frontend (Angular):**
   ```bash
   cd ../frontend/frontend
   npm install
   ng serve
   ```
   Abre tu navegador en [http://localhost:4200](http://localhost:4200)

---

## 📚 Ejemplos de uso

- **Suma binaria:** Ingresa dos números binarios y observa cómo la máquina realiza la suma bit a bit.
- **Escribir nombre:** Personaliza el texto y mira cómo la máquina lo escribe en la cinta.
